# DINING WITH ISS

## Objectives:

-   Create a fun frontend project using 2 API of your choice.

## API used in this project

-   Google maps
-   ISS coordinates

## What is dining with ISS all about?

-   Dining with ISS searches the nearest restaurant to the current location of the ISS.
